# 智海-录问-R1 推理数据（wisdomInterrogatory-R1）
## 项目概要

- 智海-录问(wisdomInterrogatory)是由浙江大学、阿里巴巴达摩院以及华院计算三家单位共同设计研发的法律大模型。核心思想：以“普法共享和司法效能提升”为目标，从推动法律智能化体系入司法实践、数字化案例建设、虚拟法律咨询服务赋能等方面提供支持，形成数字化和智能化的司法基座能力。
- 智海-录问原模型项目链接：https://github.com/zhihaiLLM/wisdomInterrogatory
- 相关报道[“智海-录问”法律大模型正式发布，行业首个法律大模型评估指标体系公开](https://mp.weixin.qq.com/s/LhXmh0_Q3s0gTruwNZ_kYQ)
- 本项目为**智海-录问(wisdomInterrogatory)**的推理强化版本，记为**智海-录问-R1(wisdomInterrogatory-R1)**。

## 智海-录问R1的优势和特色

**(1) 应对复杂法律问题：从单步判断到思维链驱动的多步推理优化**
- 传统法律大模型：难以处理需要多步逻辑推理的问题（如法律漏洞分析、矛盾条款适用、多因素权衡等）。
- 智海-录问-R1：通过思维链（Chain-of-Thought）技术，能够分解复杂问题、模拟法律推理流程（例如：事实认定→法律要件匹配→例外情形排除→结论生成），更贴近人类法律工作者的分析模式。
  
**(2) 破解黑箱困境：从隐性结论输出到可验证推理路径的范式突破**
- 传统法律大模型：直接输出结论但隐藏推理过程，用户无法验证逻辑路径，可能导致隐蔽性错误（例如错误援引失效法条或忽略关键判例）。
- 智海-录问-R1：通过显式展示推理步骤（如前提假设、法律依据、逻辑跳跃点），允许用户实时检查模型思维过程，便于发现偏差并人工干预。
  
**(3) 法律方法论融合：从通用文本生成到专业推理框架的体系化升级**
- 传统法律大模型：面对法律领域的特殊需求（如证据链分析、抗辩策略模拟、量刑情节权衡）时，生成内容可能缺乏针对性。
- 智海-录问-R1：整合三段论、类比推理、政策权衡等法律方法论。
